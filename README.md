# scrobbler

Scrobbler for YouTube Music.

Sends your YouTube Music listening history to Last.fm.

## How to run

### Local development

1. Clone the repository
2. Install dependencies:

```bash
  pnpm install
```

3. Run the development server:

```bash
  pnpm dev
```

### Production build

#### Build the project:

```bash
  pnpm build
```

#### Run the project:

```bash
  pnpm start
```
